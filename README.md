# Helpi – aplikacja dla wolontariuszy i organizacji

Helpi to nowoczesna platforma internetowa, która łączy **zweryfikowane organizacje** (NGO, fundacje, stowarzyszenia, inicjatywy lokalne) z **wolontariuszami**.
Instrukcja działania:
1. Zweryfikowane organizacje dodają ogłoszenia o pomoc.
2. Do ogłoszeń mogą zgłaszać się wolontariusze.
3. Jeżeli osoba nie może pomóc fizycznie istnieje możliwość wpłaty - wsparcia finansowego.
4. W aplikacji jest widoczna mapa z zaznaczonymi ogłoszeniami w okolicy.

---

## Tech stack

Projekt korzysta z:

- **React**
- **React Router** (`react-router-dom`) – nawigacja po aplikacji
- **Tailwind CSS** – stylowanie
- **Laravel** - backend
